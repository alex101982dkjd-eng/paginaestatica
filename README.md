# Abarrotes Store

Página estática para tienda de abarrotes con integración a WhatsApp y correo electrónico.

## Estructura del proyecto

```
abarrotes-store/
└── client/   → Frontend React (Vite) listo para Netlify
```

## Secciones
- Inicio (hero + productos destacados)
- Catálogo (por categorías)
- Ofertas y Promociones
- Nosotros
- Contacto

## Tecnologías
- **Frontend:** React + Vite
- **Despliegue:** Netlify (sitio estático)
- **Contacto:** WhatsApp API (wa.me) + mailto
